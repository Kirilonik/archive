## Основная палитра

### Фон (Apple-стиль, светлая тема)

- **Основной фон**: Градиент от светло-серого к серому
  - Верх: `#F8FAFC` (Slate-50)
  - Середина: `#F1F5F9` (Slate-100)
  - Низ: `#E2E8F0` (Slate-200)
- **Радиальные градиенты**: Мягкие белые блики с прозрачностью
  - `rgba(255, 255, 255, 0.15)` - основной блик
  - `rgba(240, 248, 255, 0.12)` - голубоватый оттенок
  - `rgba(248, 250, 252, 0.1)` - светлый блик
  - `rgba(241, 245, 249, 0.08)` - приглушенный блик

### Акцентный цвет

- **Основной акцент**: `#0A84FF` (iOS System Blue)
  - RGB: `rgb(10, 132, 255)`
  - Используется для: кнопок, ссылок, активных элементов

### Текст

- **Основной текст**: `#0F172A` (Slate-900)
  - RGB: `rgb(15, 23, 42)`
- **Приглушенный текст**: `rgba(51, 65, 85, 0.8)` (Slate-700 с прозрачностью)
  - RGB: `rgb(51, 65, 85)`

### Границы

- **Границы элементов**: `rgba(148, 163, 184, 0.3)` (Slate-400 с прозрачностью)
  - RGB: `rgb(148, 163, 184)`

### Liquid Glass элементы

- **Фон карточек**: `rgba(255, 255, 255, 0.7)` - полупрозрачный белый
- **Backdrop blur**: `blur(40px) saturate(180%)`
- **Тени**: Мягкие черные тени с низкой прозрачностью
  - `rgba(0, 0, 0, 0.08)` - основные тени
  - `rgba(0, 0, 0, 0.04)` - легкие тени
  - `rgba(0, 0, 0, 0.12)` - тени при hover

### Дополнительные цвета

- **Оценка Кинопоиска**: `#F97316` (Orange-500) с прозрачностью 90%
- **Моя оценка**: Градиент `from-blue-500 via-blue-400 to-cyan-400`
  - `#3B82F6` (Blue-500)
  - `#60A5FA` (Blue-400)
  - `#22D3EE` (Cyan-400)
- **Звездочки рейтинга**: `#F59E0B` (Amber-500)
- **Ошибки/Удаление**: `#EF4444` (Red-500)

## Стиль дизайна

- **Glassmorphism (Liquid Glass)**: Полупрозрачные элементы с размытием
- **Минимализм**: Чистые линии, много белого пространства
- **Apple-стиль**: Светлая тема с мягкими градиентами и тенями
- **Скругления**: `border-radius: 20px` для карточек, `12px` для кнопок

## Для иконки рекомендуется использовать:

- **Фон**: Градиент `#F8FAFC → #F1F5F9 → #E2E8F0` или однотонный `#F1F5F9`
- **Буква "А"**: `#0A84FF` (акцентный синий) или `#0F172A` (темный текст)
- **Эффект**: Легкий glassmorphism с размытием и мягкими тенями
- **Скругления**: 20-24px для иконки
