# Описание для генерации иконки сайта "Архив"

## Визуальное описание

**Стиль**: Минималистичная иконка в стиле Apple Liquid Glass (Glassmorphism)

**Композиция**:
- Квадратная иконка со скругленными углами (радиус ~22% от размера)
- По центру крупная буква "А" (заглавная, жирная)
- Фон с градиентом и мягкими бликами

## Цветовая палитра

### Фон
- **Основной градиент**: Вертикальный градиент от светло-серого к серому
  - Верх: `#F8FAFC` (очень светло-серый, почти белый)
  - Середина: `#F1F5F9` (светло-серый)
  - Низ: `#E2E8F0` (серый)
- **Дополнительные блики**: Полупрозрачные белые радиальные градиенты для объема
  - `rgba(255, 255, 255, 0.15)` в верхней левой части
  - `rgba(240, 248, 255, 0.12)` в верхней правой части

### Буква "А"
- **Цвет**: `#0A84FF` (яркий синий, iOS System Blue)
- **Стиль**: Жирная (Bold, 700), крупная, по центру
- **Тень**: Мягкая тень `rgba(0, 0, 0, 0.2)` с размытием 8px, смещение вниз на 4px

### Эффекты
- **Glassmorphism**: Легкое размытие фона (blur ~2px)
- **Полупрозрачный слой**: `rgba(255, 255, 255, 0.3)` поверх фона
- **Скругления**: Радиус скругления ~22% от размера иконки (112px для 512px)

## Технические детали

**Размер**: 512x512px (или кратный этому размеру для разных разрешений)

**Шрифт**: System font (San Francisco на macOS, Segoe UI на Windows, Roboto на Android)
- Семейство: `system-ui, -apple-system, BlinkMacSystemFont, 'Segoe UI', sans-serif`
- Вес: 700 (Bold)
- Размер буквы: ~55% от высоты иконки (280px для 512px)

**Пропорции буквы "А"**:
- Центрирована по горизонтали и вертикали
- Занимает примерно 50-60% высоты иконки
- Отрицательный letter-spacing для более компактного вида

## Промпт для AI-генерации

```
Создай минималистичную квадратную иконку в стиле Apple Liquid Glass:
- Фон: вертикальный градиент от #F8FAFC через #F1F5F9 к #E2E8F0
- Скругленные углы (радиус ~22%)
- По центру крупная жирная буква "А" цвета #0A84FF (яркий синий)
- Мягкие белые блики для объема
- Легкий glassmorphism эффект с размытием
- Мягкая тень под буквой
- Стиль: чистый, минималистичный, современный, в духе Apple
```

## Альтернативные варианты

### Вариант 1: Темная буква на светлом фоне
- Буква "А": `#0F172A` (темно-серый, почти черный)
- Фон: тот же светлый градиент

### Вариант 2: С акцентной рамкой
- Добавить тонкую рамку `rgba(10, 132, 255, 0.3)` вокруг иконки
- Буква остается синей `#0A84FF`

### Вариант 3: С градиентом на букве
- Буква "А" с градиентом от `#0A84FF` к `#22D3EE` (синий к голубому)
